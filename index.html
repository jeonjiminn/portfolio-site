PROJECT 01

P-Mart

React

React 기반 쇼핑몰 UI

React 기반으로 제작한 쇼핑몰 UI 프로젝트입니다. JSON 데이터를 활용하여 상품 정보를 동적으로 렌더링하고, 카테고리별 상품 리스트 및 상세 페이지 구성 등 실제 쇼핑 흐름을 고려해 UI를 구성했습니다.

✔ 주요 기능

  • JSON 상품 데이터 연동 및 동적 렌더링
  • React Router 기반 페이지 이동
  • 카드형 상품 UI 컴포넌트 구현
  • 반응형 웹 구현 및 모바일 퍼스트 구조

✔ 나의 역할

  • 전체 UI 레이아웃 및 퍼블리싱 담당
  • React 컴포넌트 구조 설계
  • 상품 데이터 매핑 및 동적 리스트 구현
  • 메인/서브 페이지 구조 및 디자인 작업

✔ 기술 스택

React, JavaScript(ES6+), JSON, CSS3
